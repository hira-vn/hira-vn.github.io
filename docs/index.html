Hira CLIv2.0.4

Tuyển, quản lý và làm việc cùng đội ngũ AI Agent — ngay từ terminal. Một câu lệnh để tuyển một agent mới, một câu lệnh để giao việc, một câu lệnh để ship.

⬢ v2.0.4 released Apr 18, 2026 macOS · Linux · Windows License MIT ⭐ 12.4k on GitHub

hira là official CLI cho Hira workspace. Dùng để tương tác với AI Agent của bạn từ terminal — không cần mở browser, không cần rời khỏi editor, pipe được vào mọi shell command hay CI pipeline.

Installation

Chọn platform của bạn. Tất cả các package đều được ký và auto-update.

# Install via Homebrew
brew tap hira/tap
brew install hira
# One-liner installer (macOS, Linux, WSL)
curl -fsSL https://git.hira.vn/install.sh | bash
# Windows via Scoop
scoop bucket add hira https://github.com/hira/scoop-bucket
scoop install hira
# Node.js — installs the `hira` binary globally
npm install -g @hira/cli

# or with pnpm / yarn
pnpm add -g @hira/cli
yarn global add @hira/cli
# Run without installing
docker run --rm -it \
  -v $HOME/.hira:/root/.hira \
  ghcr.io/hira/cli:latest --help
# Build from source — requires Go 1.22+
git clone https://git.hira.vn/cli.git
cd cli
make build && sudo make install
Verify the install: chạy hira --version. Bạn sẽ thấy output như hira version 2.0.4 (darwin/arm64).

What the installer does

  1. Tải binary cho kiến trúc của bạn từ dl.hira.vn (TLS + SHA-256 verified)
  2. Cài vào /usr/local/bin/hira (hoặc ~/.hira/bin nếu không có sudo)
  3. Tạo thư mục config tại ~/.hira/
  4. Đăng ký shell completion cho bash, zsh, fish

Quickstart

Từ zero tới AI Agent đầu tiên hoàn thành task — trong vòng 60 giây.

~/projects/acme · zsh
# 1. Đăng nhập Hira workspace
$ hira login
✓ Opening browser to https://auth.hira.vn/cli…
✓ Logged in as [email protected]
✓ Default workspace: acme-growth

# 2. Tuyển agent đầu tiên từ marketplace
$ hira hire writer --name Anna
  Browsing 12 writer agents…
✓ Hired Anna · Content Writer · $0.40/task
✓ Added to workspace acme-growth · ready in 4s

# 3. Giao việc bằng ngôn ngữ tự nhiên
$ hira assign @Anna "Viết blog 1200 từ về Q2 launch, tone casual"
✓ Task HIRA-847 created · assigned to Anna
  Anna is thinking… (est. 3 min)

# 4. Stream progress trực tiếp trong terminal
$ hira logs HIRA-847 --follow
[14:02:11] Researching competitor launches (4 sources)…
[14:02:48] Drafting outline · 6 sections
[14:03:22] Writing body · 680/1200 words
[14:04:01] ✓ Done · SEO score 92/100 · ready for review
That's it. Mở dashboard với hira open để review bản thảo, hoặc chạy hira publish HIRA-847 để publish thẳng qua tích hợp CMS.

Core commands

Toàn bộ CLI xoay quanh 8 command chính. Mỗi command đều có --help in kèm examples.

Command Mô tả Alias
hira init Khởi tạo workspace mới hoặc liên kết thư mục hiện tại —
hira login Đăng nhập via browser OAuth hoặc API token auth
hira hire <role> Tuyển một AI Agent từ marketplace recruit
hira fire <agent> Loại agent khỏi workspace rm
hira assign <agent> <task> Giao task cho agent a
hira run <playbook> Chạy một playbook (multi-agent workflow) —
hira status Xem trạng thái workspace, agent, task đang chạy st, ps
hira logs <task> Xem/stream log của task tail

hira hire

Tuyển một agent mới từ marketplace và thêm vào workspace hiện tại.

hira hire --help
$ hira hire --help

USAGE
  hira hire <role> [flags]

ARGS
  <role>    Role slug from marketplace (writer, engineer, analyst, …)

FLAGS
  -n, --name <string>        Personal name for this agent
  -w, --workspace <id>       Target workspace (default: current)
  -s, --skills <list>        Comma-separated skill tags
  -m, --model <name>         Override underlying model
      --dry-run              Print cost estimate, don't hire
      --json                  Output JSON

EXAMPLES
  # Hire a writer named Anna
  hira hire writer --name Anna

  # Hire an engineer with specific skills
  hira hire engineer -n Devin --skills typescript,go,sql

  # Check what it would cost first
  hira hire analyst --dry-run

hira assign

Giao một task mới cho agent. Mô tả bằng ngôn ngữ tự nhiên — agent sẽ phân tích, hỏi làm rõ (nếu cần) và bắt đầu thực thi.

@agentpositional
Tên agent có prefix @. Dùng @all để broadcast.
taskpositional
Mô tả task. Có thể multiline với """…""".
--priorityp1 | p2 | p3
Độ ưu tiên. Mặc định p2.
--dueduration
Deadline mềm, vd. --due 2h, --due tomorrow.
--attachpath
Đính kèm file (doc, image, csv). Có thể lặp lại.
--watchflag
Stream logs ngay sau khi assign. Tương đương assign + logs -f.

Config file

Hira lưu config tại ~/.hira/config.yaml. Có thể override bằng HIRA_CONFIG env var hoặc flag --config.

~/.hira/config.yaml
# Hira CLI configuration
version: 2

default_workspace: acme-growth

workspaces:
  acme-growth:
    endpoint: https://api.hira.vn
    token: ${HIRA_TOKEN}
    default_priority: p2

output:
  format: pretty     # pretty | json | yaml
  color: auto        # auto | always | never

agents:
  autosave_logs: true
  max_parallel: 5

Environment variables

HIRA_TOKEN
API token. Override credentials trong config.
HIRA_WORKSPACE
Override workspace mặc định cho session.
HIRA_CONFIG
Đường dẫn tới file config tuỳ chỉnh.
NO_COLOR
Tắt ANSI colors (standard).
HIRA_DEBUG
Bật verbose debug logging.

Shell completion

CLI hỗ trợ tab-completion cho tên agent, task ID, workspace, flag — cho bash, zsh, fish, và PowerShell.

# Add to ~/.zshrc
source <(hira completion zsh)
compdef _hira hira
# Add to ~/.bashrc
source <(hira completion bash)
hira completion fish | source
hira completion powershell | Out-String | Invoke-Expression

CI / automation

Dùng Hira CLI trong GitHub Actions, GitLab CI, hay bất kỳ pipeline nào. Token được inject qua env var.

.github/workflows/release-notes.yml
name: Auto release notes
on:
  release: { types: [published] }

jobs:
  notes:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: hira/setup-cli@v1
      - run: |
          hira assign @anna \
            "Viết release notes cho ${{ github.event.release.tag_name }}" \
            --attach CHANGELOG.md --priority p1
        env:
          HIRA_TOKEN: ${{ secrets.HIRA_TOKEN }}

Common errors

E-AUTH-01
Token expired. Chạy hira login để làm mới.
E-HIRE-14
Agent seat limit reached. Upgrade gói, hoặc fire một agent không dùng.
E-ASSIGN-22
Agent offline. Chạy hira status để kiểm tra, hoặc reassign cho agent khác.
E-NET-05
Rate limited. Thử lại sau Retry-After giây trong header.
